# 概览

Kubernetes被誉为云时代的操作系统，是目前云上环境中用的最为广泛的容器编排方案。作为CNCF的核心项目，其得到了非常广泛的关注，并且开源社区在其之上诞生出了非常多的开源项目，在容器编排、服务治理、DevOps等诸多领域都有着深远影响。
