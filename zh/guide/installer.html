# 安装部署

DolphinDB支持以容器形式运行，可以在Kubernetes在新窗口打开、Docker Compose在新窗口打开等容器环境中部署，也支持在亚马逊云(AWS)在新窗口打开、阿里云在新窗口打开等公有云平台中部署。

DolphinDB的容器镜像可参考DockerHub在新窗口打开。

# Kubernetes

DolphinDB支持以 Docker 作为容器运行时在Kubernetes在新窗口打开环境中。

详情参考部署DolphinDB到Kubernetes环境

# Docker Compose

DolphinDB可以部署在Docker Compose在新窗口打开环境中。

详情参考部署DolphinDB到Docker Compose环境

# 公有云平台

DolphinDB可以部署在亚马逊云(AWS)在新窗口打开、阿里云在新窗口打开等公有云中。
